Oh hi! I'm TJ VanToll. I'm a web developer, speaker, and writer and this is my blog.

I live in Lansing, MI where I work as a Developer Advocate for Telerik. I have been making web sites since the web was a thing, and have experience building everything from major applications for Fortune 500 companies to small personal sites.

I'm also an open source advocate and a jQuery UI team member. This blog itself is open source and if you'd like to dig around the source to see how I've made it you're free to.
